Whats New WPF 4.0

Whats New WPF 4.0

Date Completed: 15/08/2016

Rating: ***

Full documentation (odt format) is not available at this time

This course covers some of the new features in WPF 4.0. There are some good, reasonably detailed modules on the new Calendar controls and the DataGrid at the beginning of the course.

The Visual State Manager is now available in WPF (it was first available in SilverLight) and then later on there is coverage of Graphics, Animation and Multitouch which may be of interest.

However, the module on Text and ClearType is extremely in depth and feels a bit like the author was either interested in investigating this topic himself or wanted to demonstrate how much knowledge he had in this area. I cant really see myself using this information extensively unless I get involved in a project which requires alot of text rendering.

I didnt bother watching the Module on XBAP Script Interop as I cant imagine myself ever using the XBAP (XAML Browser Application) features of WPF.





Enterprise WPF with XAML and C# from Scratch

Enterprise WPF with XAML and C# from Scratch

Date Completed: 16/08/2016

Rating: **

Full documentation (odt format) is not available at this time

This course is meant to be aimed (at least you would this so from the title) at Enterprise Development. The first two modules cover XAML fundamentals but are pretty basic and didn't really cover anything I didn't already know. The module on Data Binding was also pretty basic.

I quite liked the module on Asynchronous Programming because, although it was still quite a basic level module, it showed a simple demo where although the processor was tied up working out Square Roots the user could still get a button to respond to a Click event. This highlighted to me the advantage of asynchronous programming quite nicely.

The Advanced Controls chapter might be worth a re-visit at some point to refresh my knowledge of any of the following controls:

  • Tab Control
  • Data Grid
  • Tree View
  • Status Bar
  • Menu Control

Similarly the modules on Animation and Styles and Templates were interesting because these are topics that I dont have much experience in yet.
